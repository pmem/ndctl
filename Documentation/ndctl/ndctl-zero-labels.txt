ndctl-zero-labels(1)
====================

NAME
----
ndctl-zero-labels - zero out the label area on a dimm or set of dimms

SYNOPSIS
--------
[verse]
'ndctl zero-labels' <nmem0> [<nmem1>..<nmemN>] [<options>]

include::labels-description.txt[]
This command resets the device to its default state by
deleting all labels.
